<h2>Usage</h2>
<ol>
  <li>Copy <code>theme.toml</code> to <code>$HOME/.config/spotify-player/</code>.</li>
  <li>Set your <code>theme</code> in <code>$HOME/.config/spotify-player/app.toml</code> to <code>Gruvbox</code> (e.g. <code>theme = "Gruvbox"</code>).</li>
</ol>
